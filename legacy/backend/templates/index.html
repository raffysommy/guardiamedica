This is the backend for the Guardiamedica shift scheduling application.

To generate a schedule, send a POST request to /generate-schedule with year and month parameters.

Example using curl:

curl -X POST "http://localhost:8000/generate-schedule?year=2026&month=1"

Or visit /docs for the OpenAPI interactive documentation.
